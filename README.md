# Financial Tracker

A simple web-based financial tracker built with **HTML, CSS, and JavaScript**.  
This project allows users to add, view, categorize, and manage expenses with persistent storage using **localStorage**.

---

## âœ¨ Features

- **Add Expenses**: Enter amount, category, and description.
- **View Expenses**: See all expenses listed with totals.
- **Reports**: Breakdown of spending by category.
- **Delete Expenses**: Remove individual entries with smooth fade-out animation.
- **Clear All Data**: Wipe all expenses from localStorage with one click.
- **Persistent Storage**: Data is saved in the browser using localStorage.
- **Animations**: Smooth transitions for page load, list items, buttons, and reports.
